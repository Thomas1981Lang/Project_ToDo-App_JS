ToDo-App
Schnittstellenbeschreibung

1.) REQUEST UserInterface 
URL: http://localhost:5000/
Method:GET
Response-Format: HTML

2.) REQUEST Neuer Eintrag/Speichern

URL: http://localhost:5000/todo
REQUEST-Methode: POST
Format Daten: POST

Daten:
{
	task: STRING
    	comment: STRING
    	option: STRING
    	date: DATE
    	prio: INTEGER
    	timecreate: INTEGER
    	timedone: INTEGER
      	done: BOOLEAN
}


Response Success
HTTPCode: 200
{"result":true}


Response Error
HTTPCode 500
